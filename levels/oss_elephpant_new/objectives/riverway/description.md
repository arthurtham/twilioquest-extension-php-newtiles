# Insert Objective Pun Here

Sometimes, it's necessary to collect values from the player that you will store and use again later, like API keys or phone number SIDs. Your validation code can both consume environment variables that have already been set, and set new ones as a result of your validation code.

Enter environment variable values in the text fields on the right. When you click `HACK`, they will be stored as new environment variables that will be available in your validation code, and in code the developer can execute in the QuestIDE.
